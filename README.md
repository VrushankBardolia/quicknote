## Getting Started
1. Clone the repository
   
   ```
   git clone https://github.com/VrushankBardolia/quicknote.git
   ```
   
2. Navigate to the project directory

   ```
   cd QuickNote
   ```
   
3. Install dependencies

   ```
   flutter pub get
   ```

4. Run the app
   ```
   flutter run
   ```
